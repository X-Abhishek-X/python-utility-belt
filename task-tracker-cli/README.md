# Task Tracker CLI 📅

> **Simple task management for the terminal.**

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

A robust command-line application built with Python to manage your daily tasks, deadlines, and priorities from the comfort of your terminal.

---

## 🚀 Usage

```bash
python task_tracker.py add "Finish cleanup" --priority high
python task_tracker.py list
```
