# ASCII Art Generator 🎨

> **Transform text into stunning terminal art.**

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

A simple Python CLI tool that converts strings into large ASCII art using the `pyfiglet` library. Perfect for terminal headers and personalized scripts.

---

## 🚀 Usage

```bash
python ascii_gen.py "Hello World"
```
