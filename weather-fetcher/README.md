# Weather Fetcher ☁️

> **A minimalist CLI tool for real-time weather diagnostics.**

[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)

A lightweight Python script that fetches current weather data for any city globally using the OpenWeatherMap API. Designed for developers who need weather status directly in their terminal.

---

## 🚀 Usage

```bash
python weather_fetcher.py "London"
```

---

## 🛠️ Installation

```bash
pip install requests
git clone https://github.com/X-Abhishek-X/weather-fetcher.git
cd weather-fetcher
```
