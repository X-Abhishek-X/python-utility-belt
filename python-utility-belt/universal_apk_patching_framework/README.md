# Universal APK Patching Framework

A modular CLI tool for patching Android APKs using plugin-based architecture.

## Features
- Drop-in APK and patch file
- Modular plugin system
- CLI interface (Python, Click/Typer)
- Subprocess management

## Getting Started
- Install requirements: `pip install click typer`
- Run: `python main.py --help`
