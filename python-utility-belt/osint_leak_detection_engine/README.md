# OSINT & Leak Detection Engine

A tool to monitor your GitHub username/email for leaks and mentions across the web.

## Features
- Monitors breaches and forums
- Integrates HaveIBeenPwned API
- Scheduled scans (cron/GitHub Actions)

## Getting Started
- Install requirements: `pip install requests`
- Run: `python main.py --help`
