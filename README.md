# Python Utility Belt 🛠️

> **A collection of high-quality, minimalist Python tools and micro-projects.**

This repository serves as a centralized hub for various utility scripts and small applications I've developed. Each sub-project is self-contained and demonstrates specific competencies in Python programming, CLI design, and automation.

---

## 📂 Project Directory

### ☁️ [Weather Fetcher](./weather-fetcher)
- **Description**: Real-time terminal-based weather diagnostics.
- **Tech**: Requests API, JSON parsing.

### 💰 [Expense Tracker](./expense-tracker)
- **Description**: Command-line financial management and categorizing.
- **Tech**: Data persistence, CLI arguments.

### 🎨 [ASCII Art Generator](./ascii-art-generator)
- **Description**: Convert text strings into terminal headers.
- **Tech**: External library integration (pyfiglet).

### 📅 [Task Tracker CLI](./task-tracker-cli)
- **Description**: Robust task management system for the terminal.
- **Tech**: Logic flow, status management.
